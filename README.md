## ✅ Rules

### 방법

1. 이 저장소를 `fork`한다.
2. 자신의 이름으로 브랜치를 만든다.
3. 생성된 저장소에 `{플랫폼}/{이름}/week{주차}_{문제번호}.{확장자}`로 파일을 생성한다.
4. 알고리즘 풀이 후 Pull Request를 보낸다.

### PR 컨벤션

- 제목: `[주차-{플랫폼}] {문제이름}`
  - ex) [week1-BAEKJOON] A+B
- 시간복잡도(필수)
- 회고(선택)
